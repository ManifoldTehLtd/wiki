# 6. 维护与故障排除

## 6.1 日常维护

1. 清洁规范：

* 使用气吹清除镜头灰尘
* ⚠️禁止使用腐蚀性溶剂擦拭窗口玻璃，腐蚀性溶剂会破坏玻璃的表面镀膜；当腐蚀性溶剂从窗口玻璃边缘缝隙进入，会腐蚀玻璃的防水背胶，导致防水失效。禁用的腐蚀性溶剂包含但不限于：酒精、异丙醇、白电油、松节油等。

2. 存储条件

* 温度：-20℃\~55℃ | 湿度：<85% RH

## 6.2 常见问题处理

| 现象        | 解决方案                    |
| ------------- | ----------------------------- |
| 点云缺失    | 检查窗口片污渍/检查镜头污渍 |
| 定位漂移    | 重启设备并静止状态初始化    |
| USB识别失败 | 设备断电重启，并重新插拔USB    |
| 升级固件| 请通过FAE提供的链接下载固件，并依据文档指引进行更新|
